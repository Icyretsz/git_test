1. Elements formatting and list

    2. a. Unordered list (ul)

    • strong: makes text bold and add importance.
    • b: only makes text bold (not recommend using).
    • em: makes text italic and add importance.
    • i: only makes text italic (not recommend using).
    • mark: marks the text.
    • small: makes smaller text.
    • del: add strike-through to text.
    • ins: underline the text.
    • sub: makes text subcript.
    • sup: makes text superscript.

    b. Ordered list (ol)

    1. strong: makes text bold and add importance.

  2. Links and images

    3. a. Links

    • The anchor elements: wrapping the text or another HTML element we want to be a link with an <a> tag.
    • HTML attribute: gives additional information to an HTML elements. It's always in the opening tag.
      1. href: anchors/links an resource to text. Remember to add rel="noopener and noreferrer" for security reasons.
      2. target: specifies where the liked resource will be opened (add _blank to open in new tab).

      Example: <a href="https://www.facebook.com/biglak1" target="_blank" rel="noopener noreferrer">My Facebook page</a> will give My Facebook page

    • Absolute links: links to other websites. (protocol://domain/path)
    • Relative links: links to other pages within our own website.

      • Example: <a href="./pages/about.html">About</a> will give About (By adding ./ you are specifying to your code that it should start looking for the file/directory relative to the current directory).

      When use href to link a website to text, the website link must be typed in full (https://www.website.com), if https is ommitted, the browser will try to look inside of ur computer instead of on the internet.
